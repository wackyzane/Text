Random Dad Jokes

3.0: We made a random generator for dad jokes. For the beginning we call in the class app and extend it to react.component so that we can use the identifier 'this'. We use super(); to actually call out the function that allows for 'this'. We then set joke to null and isfetchingjoke to false. This is so we can't have multiple requests coming in at the same time. After that we bind onSearchChange and searchJokes so that we can callback the function without it losing its meaning. That is everything for the constructor.

3.2: Next we start with the functions. The first is searchJokes which we limit the amount it shows to 20. We also setState of isFetchingJoke to true so that people can't keep pressing the button and spamming the requests to look for jokes. WE now use fetch to grab the jokes the the website icanhazdadjoke.com and set it to the limit we stated (20). Now we need to set the method which determines waht we are doing on the site. This is told on the site to get the jokes. We set method to "GET". we also made header and put Accept: to "application/json". This is for the same reason as method. At the end of fetch we put what it will do with the jokes after the program has gotten them. We put all the jokes into the array jokes and set isFetchingJoke to false.

3.4: This is a small method that changes the searchTerm to the value we give it. We will give it a value in a different method.

3.6: This is also a somewhat small method. This method grabs the jokes we just got and sets them in a unordered list with each of the jokes being an item in that list.

3.8: Last is the render function. In the return we put our div with class App so we have customization. Next we call in out JS with searchForm. In this call we also set all variables, like if it is searching or if you only want one search result. I will explain searchForm result in section 4. Under our search bar and buttons we have it says if it is searching results if isFetchingJoke is true. Then we render the jokes. At the end of the page we have render correctly and load the page.
